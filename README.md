This is a demo implementation of a Lead Intelligence Agent.

Data Sources:
- PubMed (live scientific publication data via NCBI Entrez)

Notes:
- LinkedIn, funding, and technographic signals are simulated or inferred
  due to API and ToS limitations.
- Email generation is heuristic.
- The goal is to demonstrate pipeline logic, scoring methodology,
  and decision-making alignment with business development workflows.

This prototype implements:
 Identification
 Enrichment
 Probability Ranking
